This package contains two binaries:
- demo_<n>_openal: OpenAL build
- demo_<n>_applaudio: applaudio build

Run the binary that matches the audio backend you want to test.

To run either executable, you might have to first do:
  chmod ugo+x demo_<n>_openal
 or:
  chmod ugo+x demo_<n>_applaudio

In the hosted Linux release bundle, choose the modern or legacy folder that matches your system before running the demo.
